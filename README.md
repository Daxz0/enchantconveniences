# About
Inspired from **highkeyoli**, chiseled bookshelves can now power enchanting tables!

# Features
- Enchanting tables no longer need line of sight with bookshelves for power
- You can use chiseled bookshelves now instead of bookshelves
- Chiseled bookshelves must be **FULL** (6 books) in order to act like a normal bookshelf.
- Enchanted books inside of chiseled bookshelves now have a higher chance of showing up when enchanting books. (Requires 3 books for +1 additional chance, and 5 **mending** books for +1 additional chance)
- Enchanting tables can now receive power from bookshelves from a 5 block range.




Claude was used to debug errors with the code.
